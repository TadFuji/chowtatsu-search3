# 調達の羅針盤 (Chowtatsu Search 3)

「調達の羅針盤」は、東京都、神奈川県、そして政府ポータルの入札情報を一括検索できるアプリケーションです。
生成AIを活用し、ユーザーが入力した自然言語（例：「魚屋の仕事がしたい」）を、お役所言葉（例：「水産物」「物品購入」）に自動変換して検索を行うのが特徴です。

## 特徴
*   **AIキーワード変換**: 専門用語がわからなくても、普段の言葉で検索可能。
*   **複数サイト横断検索**: 政府ポータル、東京都、神奈川県に対応。
*   **AI思考プロセス可視化**: AIがどのようなキーワードで検索しているかをリアルタイムで表示。

## 動作環境
*   Windows
*   Python 3.x
*   Google Chrome (Selenium利用のため)

## 起動方法
フォルダ内の `start.bat` をダブルクリックしてください。
バックエンドサーバーが立ち上がり、自動的にブラウザが開きます。

詳しい使い方は [取扱説明書.md](./取扱説明書.md) をご覧ください。

## 構成
*   `backend/`: Pythonによる検索エンジン・スクレイピング処理
*   `frontend/`: 検索用Webインターフェース
